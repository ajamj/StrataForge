Flexible Theming System

How should the UI colors be managed for future theme support?

A

Semantic Token System (Recommended)

Components use names like ActivityBar.Background instead of hex codes. Allows swapping entire palettes (Dark, Light, Solarized) easily.

B

Preset Palette Selection

Hardcoded sets of colors for 3-4 specific themes. Simpler to implement but less flexible for custom community themes.

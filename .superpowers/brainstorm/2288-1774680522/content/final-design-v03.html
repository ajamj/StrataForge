v0.3 Design Summary: Interactive interpretation

From data viewing to active geological interpretation.

Key Features

  • Seed & Auto-Track: Click a reflector, and the app tracks the event across the slice.
  • Smooth Surfaces: Turn sparse picks into continuous geological horizons using RBF interpolation.
  • High-Performance I/O: Memory-mapped SEG-Y access for near-instant slice extraction.
  • Picking Workflow: Integrated tools to create, edit, and manage multiple interpretation horizons.

Technical Focus

  • Implementing robust 2D/3D interpolation algorithms in seisly_compute.
  • Refactoring seisly_io to use memmap2 for large seismic files.

Interpretation Loop

Slice View ➔ Pick (Seed) ➔ Auto-Track ➔ Generate Smooth Surface ➔ 3D Visualization

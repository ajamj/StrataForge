Surface Generation Approaches

How should we turn sparse 2D picks into a 3D surface?

A: Delaunay Triangulation
📐

Connects picks directly with triangles.

  • Exact (passes through picks)
  • Fast computation
  • Can be jagged/angular
  • No extrapolation
B: RBF Interpolation (Smooth)
🌊

Creates a continuous mathematical surface.

  • Looks "geological"
  • Handles irregular spacing
  • Computationally expensive
  • May "overshoot" in gaps
A

Triangulation (Mesh)

Focus on precision and speed. Use the existing sf_compute triangulation logic.

B

Interpolation (Smooth)

Focus on visual quality. Implement a Radial Basis Function (RBF) or Kriging approach.

# StrataForge Library Documentation

This directory contains detailed documentation for the primary third-party libraries used in the StrataForge project.

## UI & Rendering
- [egui / eframe](./egui_eframe.md): Immediate-mode UI framework.
- [wgpu](./wgpu.md): Modern 3D rendering engine.

## Data I/O & Communication
- [memmap2](./memmap2.md): High-performance memory-mapped file I/O.
- [tonic / prost](./tonic_prost.md): gRPC and Protocol Buffers for AI microservice communication.

## Mathematics & Algorithms
- [nalgebra](./nalgebra.md): Linear algebra for transformations and surface interpolation.
- [spade](./spade.md): Delaunay triangulation for mesh generation.
